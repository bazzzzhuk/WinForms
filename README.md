В операционной системе Windows есть ключи реестра, связанные с планировщиком заданий (Task Scheduler). Эти ключи хранят данные о запланированных задачах, которые выполняются по заранее заданному триггеру (время, наступление события и т. п.). 

В Windows настроить запуск компьютера по времени можно через планировщик заданий. Для этого нужно создать задание, которое будет выполнять пробуждение компьютера из сна или гибернации.
